# Описание проетка

*"Это программа на Python с графическим интерфейсом (Tkinter), которая работает как генератор случайных цитат.

# Основные функции:
Конвертация между любыми валютами мира
Актуальные курсы через API exchangerate-api.com
Автоматическое сохранение истории конвертаций
Просмотр истории операций в таблице
Валидация вводимых данных (только положительные числа)
Интуитивно понятный графический интерфейс
Очистка истории и полей ввода.
